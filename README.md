# About

Welcome to the Pedestal Mod, A small mod that adds a Pedestal to display items on!
Craft the Pedestal with 7 pieces of smooth stone. Use the Pedestal to display any item you wish.

# Setup

Version: Minecraft Fabric 26.1

Fabirc API is required
